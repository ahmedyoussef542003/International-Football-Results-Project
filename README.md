# International Football Results Project

## Introduction
This project explores international football results data to uncover trends in hosting cities, team performance, and match outcomes using SQL and Power BI.

## Main Objectives
1. Identify which cities have hosted the largest number of international matches.
2. Determine if home teams generally have a higher chance of winning.
3. Identify which tournaments have the highest percentage of matches ending in a draw.
4. Determine which teams score the most goals and which teams concede the most goals on average.
5. Find the matches with the highest total number of goals.

## Dataset Overview
- Over 40,000 match records
- 9 main columns: date, home_team, away_team, home_score, away_score, tournament, city, country, neutral
- Dataset is included in this repository.

## Tools Used
- SQL
- Power BI

- `dataset/` → contains the dataset CSV file
